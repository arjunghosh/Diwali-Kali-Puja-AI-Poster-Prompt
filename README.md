## ✨ Features

- 🚀 **Immediate Start**: Automatically begins Q&A when file is loaded
- 🎨 **Platform Agnostic**: Works on ChatGPT, Gemini, and Copilot
- 🌏 **Bilingual Support**: Auto-adds Hindi (शुभ दीपावली) and Bengali (শुभ কালী পূजा)
- 🛡️ **Input Validation**: 50-character limit with smart validation
- ⚡ **Ultra-Optimized**: 52% size reduction from original
- 🎯 **Smart Defaults**: Culturally appropriate color palettes
- 🏢 **Multi-Identity**: Support for individuals, companies, teams, and anonymous

## 🚀 Quick Start

1. **Copy the script**: Copy `Diwali-AI-Poster-Creation-Prompt-v4.md`
2. **Paste into AI**: Paste into ChatGPT, Gemini, or Copilot
3. **Answer 8 questions**: Platform → Festival → Identity → Colors → Style → Regional → Signature
4. **Get your poster**: Receive high-resolution festival poster

## 📋 How It Works

### Q&A Flow
1. **Platform Selection**: Choose ChatGPT/Copilot (auto-generate) or Gemini (2-step)
2. **Festival Choice**: Diwali only, Kali Pujo only, or Both
3. **Identity**: Person name, Anonymous, or Company/Team
4. **Primary Color**: Recommended traditional colors or custom
5. **Accent Colors**: Complementary accent color combinations
6. **Art Style**: Traditional, Modern, Watercolor, Mandala, or Custom
7. **Regional Style**: Optional regional art styles (Bengali Alpana, Pattachitra, etc.)
8. **Signature**: Name, message, none, or custom phrase

- **Accents**: Glittering Gold/Silver, Bright Orange/Fuchsia

#### 🎊 Both Festivals

## 🎯 Best Practices

### For AI Implementers
- Use `{VARIABLE}` format for substitutions

- Enforce 50-character validation
- Update titles at specified points
